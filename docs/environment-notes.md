# 環境固有の留意事項

開発環境・実行環境に起因する問題と対策を記録する。
技術的なブロッカーではなく「知っていれば回避できる」類の知見を蓄積する。

## 目的

- 環境差異によるハマりどころを事前に共有する
- 同じ問題の再調査を防ぐ
- CI 環境とローカル環境の差異を明文化する

## 記録フォーマット

| 問題 | 原因 | 対策 |
|------|------|------|
| 問題の概要 | 根本原因 | 回避策・恒久対策 |

---

## Python 環境

| 問題 | 原因 | 対策 |
|------|------|------|
| **NDLOCR-Lite の依存関係が重い** | PyTorch + ONNX Runtime 等の大規模パッケージ | 開発用と推論用の依存グループを分離する |

## ブラウザ印刷環境

| 問題 | 原因 | 対策 |
|------|------|------|
| **「ページに合わせて縮小」で座標がずれる** | ブラウザの印刷ダイアログのデフォルト設定 | 印刷ガイドで「100%/実寸」を案内。可能なら縮小率検出 |
| **CSS mm 単位のブラウザ差異** | ブラウザ実装による微差 | 印刷検証フェーズで実測し補正テーブルを検討 |

## CI 環境（Linux）固有

| 問題 | 原因 | 対策 |
|------|------|------|
| **ローカルでは通る lint が CI で失敗** | CI はパッケージ全体をスコープとするが、ローカルでは変更ファイルだけ検証しがち | ローカルでも CI と同じスコープで lint を実行する |

## Windows 固有

| 問題 | 原因 | 対策 |
|------|------|------|
| **Makefile が動かない** | Unix シェル構文（`[ -f ... ]`, `find` 等）が Windows 非互換 | 個別コマンドを直接実行するか、WSL / Git Bash を使う。詳細は `docs/windows-setup.md` |
| **PowerShell で `.venv\Scripts\Activate.ps1` が拒否される** | 実行ポリシーがデフォルトで Restricted | `Set-ExecutionPolicy RemoteSigned -Scope CurrentUser` で解除 |
| **C 拡張パッケージのビルド失敗** | Visual C++ Build Tools 未インストール | Build Tools for Visual Studio をインストール |
