# マイルストーン（Milestones）

プロジェクトの大きな節目（マイルストーン）を定義し、完了条件を明確にするドキュメント。

## PROGRESS.md との関係

`PROGRESS.md` はプロジェクト全体の **サマリー層**（状態と差分の概況）。
`docs/milestones.md` は **詳細層**（完了条件と達成基準の定義）。
