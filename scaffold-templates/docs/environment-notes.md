## Windows 固有

| 問題 | 原因 | 対策 |
|------|------|------|
| **ビルドツールの UTF-8 パニック** | Windows コンソールのデフォルトエンコーディングが非 UTF-8 | `chcp 65001`（UTF-8 切替）+ `NO_COLOR=1`（カラー出力抑制）のラッパースクリプトを用意 |
| **開発サーバーのファイルロックエラー（EBUSY）** | 並列 dev サーバー起動時のファイルロック競合 | ブラウザリロードで回復。ウイルス対策ソフトの除外設定で軽減 |
| **CJS/ESM 相互運用の問題** | モノレポの symlink と bundler の相互作用 | `commonjsOptions.include` + `optimizeDeps.include` で明示的に指定 |

## CI 環境（Linux）固有

| 問題 | 原因 | 対策 |
|------|------|------|
| **ローカルでは通る lint が CI で失敗** | CI はパッケージ全体をスコープとするが、ローカルでは変更ファイルだけ検証しがち | ローカルでも CI と同じスコープ（パッケージ全体）で lint を実行する |
| **lint エラーの逐次修正による push 回数の増大** | エラーを1件ずつ修正 → push を繰り返す | まずエラーの全量を確認し、一括で修正してから push する |
